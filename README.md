## 文件说明
- **`resource/`**: 
  - **`Sleep.jsx`**: 睡眠页面主组件
  - **`SelectDropdown.jsx`**: 用于切换不同展示视图的下拉选择器。
  - **`SleepSummary.jsx`**: 用于展示睡眠统计数据的组件。


- **`resource/charts/`**:
  - **`stages/`**: 展示 "Stages" 图表的相关组件, 包括不同时间范围（天、周、月）。
  - **`amounts/`**: 展示 "Amounts" 图表的相关组件。
  - **`comparisons/`**: 展示 "Comparisons" 图表的相关组件。
  - **`Charts.jsx`**: 图表的主入口文件。
  - **`data/`**: 数据文件夹，包含图表所需的静态数据。
